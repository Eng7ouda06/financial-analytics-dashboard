# Financial Analytics Dashboard

An interactive dashboard simulating a real-time payments intelligence platform, built with Python, Streamlit, and Plotly.

## Overview

This project visualizes key financial KPIs across a 24-month period, including revenue trends, transaction volumes, churn rates, and fraud monitoring. It includes a 3-month revenue forecast using linear regression and interactive controls to filter and explore the data.

## Features

- Revenue trend with 3-month linear regression forecast and confidence band
- Payment category breakdown across E-commerce, Retail POS, Mobile Pay, Subscriptions, and International
- Customer churn rate monitoring over time
- Fraud rate tracking with configurable alert threshold
- Monthly transaction volume analysis
- Interactive sidebar controls — adjust time range, toggle forecast, filter categories
- Raw data table with CSV export

## Technologies

- Python, Streamlit, Plotly
- Pandas, NumPy, scikit-learn
